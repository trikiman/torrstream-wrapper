# Requirements (Archived): TorrStream v2.1 Player UX + iOS readiness

**Defined:** 2026-05-12
**Shipped:** 2026-05-12

## Player

- [x] **PLAY-03**: Vidstack is the web player; Plyr and its CDN scripts are removed from the shell.
- [x] **PLAY-04**: Audio plays by default on desktop Chrome/Firefox and on iOS Safari once the user initiates playback. No silent-playback regressions.
- [x] **PLAY-05**: Double-tap on the left third of the player rewinds 10s; double-tap on the right third fast-forwards 10s. Tap in the middle toggles play/pause.
- [x] **PLAY-06**: Player exposes the underlying HTMLVideoElement so `static/lampa-sync.js`'s `findVideo()` continues to locate it. Plugin must not need changes.
- [x] **PLAY-07**: Playback controls (play, pause, seek, volume, fullscreen, PiP where supported) work on iOS Safari and iPad Safari; inline playback is preserved (no forced fullscreen on `<video>` tap).

## Delivery

- [x] **DELV-06**: Bundle footprint increase documented (Plyr was ~120 KB JS; Vidstack is ~250 KB gzip). Loaded from CDN.
