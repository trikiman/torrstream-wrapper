<objective>
Make watch progress follow the user from Lampa to the wrapper (and back) without depending on Lampa.cloud or any third-party sync service.

Purpose: Close SYNC-03/SYNC-04/SYNC-05 by combining a wrapper-side fallback that surfaces externally-watched torrents with a Lampa-side plugin that round-trips playback time.
Output: Backend `viewed_in_torrserver` flag, relaxed continue-row filter, CORS on `/api/position/*`, and a self-installable Lampa plugin at `static/lampa-sync.js`.
</objective>

  <action>Ship a self-contained Lampa plugin that subscribes to `Lampa.Listener.follow('player', ...)` and wraps `Lampa.Player.play`. Allow overriding the wrapper URL via `Lampa.Storage.set('torrstream_sync_url', ...)`. Default the wrapper URL to `https://tv.trikiman.shop`.   </acceptance_criteria>
